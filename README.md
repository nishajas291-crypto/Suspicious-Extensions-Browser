 Task Description
This task focuses on identifying, analyzing, and removing suspicious or unnecessary browser extensions to improve browser security and performance. Browser extensions often request extensive permissions, which can pose security and privacy risks if misused or malicious.

---

 Objective
The objective of this task is to:
- Identify potentially harmful or unnecessary browser extensions
- Understand the security risks associated with browser extensions
- Remove suspicious extensions to ensure safe and secure browsing practices

---

 Tools Used
- Mozilla Firefox Browser  
- Google Chrome Browser  
- Microsoft Edge Browser  

---

 Task Procedure

### Step 1: Review Extensions in Mozilla Firefox
The Firefox browser was opened and the **Add-ons and Themes → Extensions** section was accessed.  
All installed extensions were reviewed by analyzing their purpose, permissions, and trust level.

**Observed Extensions:**
- FoxyProxy  
- Notefox: Website Notes  

No suspicious behavior or unnecessary permissions were identified.

---

### Step 2: Review Extensions in Microsoft Edge
The Microsoft Edge Add-ons section was checked to review installed and recommended extensions.

- Only trusted extensions were displayed
- No suspicious or unknown extensions were found

---

### Step 3: Review Extensions in Google Chrome
The **Chrome Extensions → Manage Extensions** page was opened to analyze installed extensions.

**Observed Extensions:**
- LinkedIn Extension  
- Google Docs Offline  

---

### Step 4: Identification of Suspicious Extension (Educational Purpose)
For learning and internship demonstration purposes, the **LinkedIn Extension** was selected for deeper analysis.

**Reasons:**
- Runs continuously in the background  
- Has access to browsing-related activity  
- Not frequently used  

---

### Step 5: Removal of Suspicious Extension
The LinkedIn Extension was removed to demonstrate the process of eliminating unnecessary or potentially risky extensions.

**Steps Followed:**
- Clicked on Remove  
- Confirmed the action  
- Restarted the browser  

---

 Step 6: Final Verification
After restarting the browser:
- No suspicious extensions were present  
- Browser performance remained stable  
- Only trusted extensions were active  

---

 Summary of Findings

|----------------|---------------------------------------------|---------------------|

Key Concepts Learned
- Browser extensions can pose serious security threats
- Excessive permissions can lead to data exposure
- Malicious extensions may track user activity or steal sensitive information
- Regular auditing of browser extensions is essential for cybersecurity

 Outcome
